# 📊 Marketing Command Center

A unified digital marketing dashboard combining campaign performance tracking,
content planning, and SEO monitoring — built to demonstrate how these three
pillars connect in a real marketing operation.

## What it does

**Overview** — At-a-glance view of total spend, revenue, blended ROAS, this
week's scheduled content, and which keywords are moving up or down in rankings.

**Campaign Dashboard** — Log campaigns by channel (Instagram, Google Ads,
Email, Facebook, LinkedIn, TikTok) and instantly see calculated KPIs:
- CTR (click-through rate)
- CPC (cost per click)
- CPA (cost per acquisition)
- ROAS (return on ad spend)

Includes a channel comparison chart so you can see which channels are
actually performing.

**Content Calendar** — Plan posts and campaigns on a visual monthly calendar,
optionally linked to a specific campaign.

**SEO Tracker** — Track keyword rankings and estimated traffic over time,
with a rank trend chart (lower rank = better) and automatic "movers"
detection showing which keywords are climbing or falling fastest.

## Why this project

Marketing decisions live and die by data. This tool shows the ability to:
- Understand core marketing metrics and how they're calculated
- Think about campaigns, content, and SEO as connected systems, not silos
- Turn a real-world workflow into a usable tool

## Tech
- HTML5, CSS3, vanilla JavaScript (no framework)
- Chart.js for data visualization
- Data persists in browser local storage — no backend required

## Running it
Just open `index.html` in a browser, or visit the deployed GitHub Pages link.
